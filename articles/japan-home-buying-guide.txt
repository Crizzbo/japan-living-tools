<p>日本对外国人购房<strong>没有国籍限制</strong>——无论你持什么签证，甚至无在留资格的外国投资者也可以买。这和很多国家（比如澳洲、新西兰）完全不同。但<strong>能买房≠能贷款</strong>，贷款才是外国人买房的真正门槛。</p>

<h2>一、外国人购房限制与误区</h2>
<h3>可以买什么？</h3>
<ul>
  <li>✅ 土地（所有权）：日本是<strong>土地永久产权</strong>（不同于中国的70年使用权）</li>
  <li>✅ 公寓（区分所有）：购买的是专有部分+共用部分的持分权</li>
  <li>✅ 投资物件：空室出租、民宿（需取得旅馆业法许可/特区民泊认定）</li>
</ul>

<h3>哪些不行？</h3>
<ul>
  <li>❌ 农地（需农业委员会许可）</li>
  <li>❌ 购买房产自动获得签证（没有房产签证这个类别）</li>
  <li>❌ 以他人名义购买规避贷款限制（违法）</li>
</ul>

<h2>二、购房全流程 6 步走</h2>

<h3>Step 1: 预算评估与贷款预审（1~2周）</h3>
<p>先去银行做<strong>事前审查（仮審査）</strong>。日本银行审查时主要看：在留资格（永住者最优）、年收入（还款额≤年收30%）、勤务年数（同一家公司3年以上有利）、首付比例。</p>
<p>永住者贷款选择最多：三菱UFJ、三井住友、りそな等都市银行利率低至 0.3%～。非永住者（技术签证等）可选中国银行东京分行、交通银行东京分行，利率约1.5~2.5%，但通常要求首付30%以上。</p>

<h3>Step 2: 物件搜索与看房（2~4周）</h3>
<p>推荐平台：SUUMO、HOME'S、at home（三大不动产门户）、LIFULL。注意：网上"已售"或"交涉中"标记不一定及时更新。实际看房建议先在Google Map上查看周边环境（便利店、超市、医院、学校、公园距离），查<strong>ハザードマップ（灾害风险地图）</strong>确认水灾·土砂灾害风险。</p>

<h3>Step 3: 购买申请与价格交涉（1周）</h3>
<p>通过中介提交<strong>「買付証明書」（购买意向书）</strong>。新房通常按定价走，几乎没有议价空间。二手房可参考公示价格和周边成约案例进行交涉。注意：物件如果已经有他人提前提交买付，卖家优先与先提交者交涉。</p>

<h3>Step 4: 签约（売買契約）（1天）</h3>
<p>签约前必做：<strong>「重要事項説明」（重要事项说明）</strong>，由宅建士（宅地建物取引士）朗读约30~60分钟的法律文件，内容包括土地权利、城市规划限制、修缮计划、管理费滞纳状况等。签约时支付<strong>手付金（定金）</strong>，通常为房价的5~10%。</p>

<h3>Step 5: 贷款本审与资金准备（1~2周）</h3>
<p>签约后向银行提交<strong>本审</strong>申请，需提供住民票、纳税证明、源泉征收票等。本审通过后银行发放<strong>金消契约（金銭消費貸借契約）</strong>。卖方、买方、银行三方协调决算日期。</p>

<h3>Step 6: 决算引渡（決済・引渡し）（1天）</h3>
<p>司法书士主持决算程序：贷款资金到位→尾款支付→登记（所有权转移·抵押权设定）→钥匙交付。这一天你会收到：権利証（登记识别情报）、建物図面、各种说明书。</p>

<h2>三、税费全拆解</h2>

<h3>签约时（税金先払い）</h3>
<ul>
  <li><strong>印纸税</strong>：买卖合同贴印花，房价1,000万~5,000万 → 1万日元（现行减免），5,000万~1亿 → 3万日元</li>
  <li><strong>登录免许税</strong>：所有权转移登记时缴纳，建筑物评估值×2%</li>
</ul>

<h3>决算时</h3>
<ul>
  <li><strong>不动产取得税</strong>：购买后约6个月~1年由都道府县寄来纳税通知书。评估值×3%（2027年3月前住宅用地评估值减半）。新房有大幅减免。</li>
</ul>

<h3>每年缴纳</h3>
<ul>
  <li><strong>固定资产税</strong>：评估值×1.4%，分4期缴纳</li>
  <li><strong>都市计画税</strong>：评估值×0.3%（市街化区域内）</li>
</ul>

<h2>四、中介费结构</h2>
<p>法定上限：<code>房价×3.3% + 6.6万日元（含消费税）</code>。例如：</p>
<ul>
  <li>房价 2,000万 → 中介费上限：72.6万日元</li>
  <li>房价 3,000万 → 中介费上限：105.6万日元</li>
  <li>房价 5,000万 → 中介费上限：171.6万日元</li>
</ul>
<p>新房有时不收中介费（开发商直销）。</p>

<h2>五、贷款全接触</h2>
<h3>Flat35（长期固定利率）</h3>
<p>住宅金融支援机构提供的全期间固定利率贷款，最长35年。2025年6月利率约1.8～2.1%。适合不想承担利率变动风险的人。</p>

<h3>変動金利（浮动利率）</h3>
<p>都市银行的浮动利率约0.3~0.6%（永住者），每半年重新评估。目前处于历史低位，但未来可能随日银政策调升。</p>

<h3>贷款所需条件（概略）</h3>
<ul>
  <li>年收300万日元以上（部分银行200万以上可）</li>
  <li>同一公司勤务1年以上（3年以上最佳）</li>
  <li>返済负担率 ≤ 30%（年还贷额 ≤ 年收×30%）</li>
  <li>永住者 → 首付最低0%（100%贷款），但利率略高</li>
  <li>技术签证 → 首付通常30%以上，且需在日居住2~3年以上</li>
</ul>

<h2>六、买 vs 租：简单决策框架</h2>
<p>如果你预计在日本生活 <strong>7年以上</strong>，有稳定的工作和首付能力，购房通常比租房划算。7年以内的话，租房灵活性更高。用我们网站的 <a href="/tools/rent-vs-buy">买房vs租房计算器</a> 输入具体数据即可直观看到累计支出对比。</p>

<p>房屋不同于消费品——它是资产。在日本以极低利率贷款，把租金变成资产积累，是多数在日华人最终选择买房的核心逻辑。</p>
