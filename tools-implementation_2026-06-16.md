# 计算器工具实现

## 时间
2026-06-16 22:17-22:27 GMT+9

## 概述
实现了三个计算器工具页面，从占位符变为可交互的功能页面。

## 1. 税后工资计算器 (`/tools/salary-calculator`)
- **数据源**: 国税厅 No.1410（給与所得控除）、No.2260（所得税率）、No.1199（基礎控除）
- **令和7年（2025年）基準**
- **功能**: 滑块调节年収 → 自动计算手取额
- **计算链**: 給与所得控除 → 社会保険料(15.65%/16.45%) → 基礎控除 → 所得税(7段階累進) → 復興特別所得税(2.1%) → 住民税(10%) → 手取额
- **输出**: 月收手取额(大字) + 年間/月額明细表
- **支持**: 40歳以上介護保険料开关

## 2. 高度人材ポイント計算器 (`/tools/hs-point-calculator`)
- **数据源**: 出入国在留管理庁 高度専門職ポイント制
- **评分维度**: 学歴(0-30) / 職歴(0-25) / 年収(0-55) / 年齢(0-15) / 日本語(0-15)
- **加点项目**: 日本大学院修了(+10) / 日本大学卒業(+10) / 世界TOP大学(+10) / 日本語+学歴(+5) / 国家資格(+5) / 研究実績(+15) / 特許(+15) / 役員経験(+20) / 政府PJ(+10) / 投資(+5)
- **判定**: 80点以上(緑) / 70-79点(青) / 未達(灰)
- **输出**: 合計点(大字) + 内訳表 + 永住申請可否

## 3. 买房 vs 租房对比 (`/tools/rent-vs-buy`)
- **功能**: 双栏参数输入 + 累計比較图 + 年度表
- **購入侧**: 物件価格/頭金/金利/ローン年数/固定資産税/修繕費/諸費用
- **賃貸侧**: 月額家賃/上昇率/更新料
- **计算**: 元利均等返済 → 各年の利息・元本・税金・修繕費累計
- **可视化**: 横向条形图对比购/租累计支出
- **输出**: 判定バナー（数値つき）+ 年度別詳細表

## 技术细节
- 全页面 `'use client'` (需交互状态)
- TypeScript 类型完整
- 使用 useMemo 优化计算性能
- 无第三方依赖，纯手写计算逻辑
